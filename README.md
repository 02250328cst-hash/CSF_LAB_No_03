aim:To understand well the concepts of Python control structures through application of the conditional statement, loops, use of break and continue, coupled with learning to define and call functions in solving problems in a structured and efficient way.
 List of Programs
1. Determine whether a number is negative, positive or zero.  
2. Determine whether a number is even or odd.  
3. Determine the greatest between two numbers.  
4. Determine the largest out of three numbers with nested if-else.  
5. Accept marks and show grade.  
6. If-elif menu driven calculator.  
7. A for loop to print out numbers form 1 to 10.  
8. Write down all those numbers even (we have 1 to 20).  
9. Add up the first 10 natural numbers.  
10. Show the multiplication table of a particular number.  
11. Write out numbers 1-5 in a while loop.  
12. Determine the number between 1 and n with a while loop.  
13. Keep on taking input until a user enters 0.  
14. Print numbers one to ten but break when the number is 6.  
15. Print out 1 to 10 without 5 continue  
16. Find a number in a list and break when the number is found.  
17. Declare and invoke a procedure which will print out Hello, World!  
18. Define a function which gives the square of a number.  
19. Specify a two-number addition function.  
20. Write a function to determine whether a number is odd or even.  
21. Write a procedure that computes factorial of a number.  
22. Write a procedure that will print the multiplication table of a number.  
23. Print out numbers between 1 and n and show whether it is even or odd with the help of a function and a loop.  
24. Function controlled calculator with menufunction and looping until the user exits.

Task 1-3: Simple conditional statements and the number property checking, and maximum number.

Task 5: Conditional statements to grade marks based on conditional statements.  
Task 6 and 24: Menu-driven calculators to prove the use of if-elif and functions.
